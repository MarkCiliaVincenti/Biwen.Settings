# Biwen.Settings

## 开源动机
- 由于当前有好几个小项目,且每个小项目都有比较多的配置项,所以把这个模块抽离出来打包到Nuget共用.
- 每个小项目以及模块可以在任意想要动态配置的地方定义Setting, 系统会直接注入到自己的业务代码中且存储到持久层 所见即所得.我感觉有部分用户有这样的需求.

## NuGet 包

- dotnet add package Biwen.Settings
